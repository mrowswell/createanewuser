<h1>Create a New User</h1>

<h2>Description</h2>
Step by step demonstration how to add a user on Active Directory.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Diskpart</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)

<br />
